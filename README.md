App en Kotlin con JPC que muestra una base de datos de chistes, alojada en un server a la que nos conectamos a través de una api. 
La app dispone de Login y Register aunque son inecesarios ya que no afecta en nada a la funcionalidad. 
Los chistes se organizan por categorías y a cada chiste podemos asignarle una puntuacion de 0 a 5 estrellas, que también se guarda en la bdd.
